# Character Counter

## Objetivo
Replicar visualmente la interfaz Character Counter utilizando únicamente HTML y CSS.

## Tecnologías
- HTML5
- CSS3
- Flexbox
- CSS Grid

## Estructura HTML
- Header
- Main
- Textarea
- Checks
- Tarjetas de métricas
- Letter Density

## CSS
Se utilizaron:
- Variables CSS
- Flexbox
- Grid
- Hover
- Box Shadow
- Media Queries

## Dificultades encontradas
- Ajustar las imágenes de ícono y tarjetas
- Utilizar id para estilización
- Crear las barras de progreso de letras
- Dimensionar apropiadamente según el ejercicio dado
